# WeKit

WeKit 修改版

## 修改内容

- 添加 WAuxiliary 与 NewMiko 目前公开源代码中的部分功能
- 移除全部校验，减少模块体积，避免不必要性能开销
- 一些功能
- 无须关闭 `Xposed API 调用保护`
- 移植部分 UI 至 Jetpack Compose

## 特色功能

- 自动回复
